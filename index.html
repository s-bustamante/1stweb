HTML

HTML es un lenguaje de marcado que se utiliza para crear paginas web. Es el lenguaje de marcado estandar para la creacion de documentos de la World Wide Web. HTML se compone de una serie de elementos que se utilizn para estructurar y presentar el contenido de una pagina web. estos elementos incluyen encabezados, parrafos, enlaces, imagenes, tablas y muchos otros. HTML es un lenguaje muy flexible y se puede utilizar para crear una amplia variedad de sitios web, desde simples paginas estaticas hasta aplicaciones web complejas.
